## ✨ Tentang Fishing OS
Fishing OS adalah proyek sistem operasi berbasis Linux yang dikembangkan untuk tujuan:
- Eksperimen personal
- Pembelajaran sistem operasi
- Otomatisasi tool yang sering digunakan dalam dunia rekayasa keamanan
